    Code Blocks

    Material for MkDocs renders beautiful, feature-rich code blocks out of the box. With a few extensions enabled you get syntax highlighting, line numbers, annotations, and copy buttons.

    Configuration

    theme:
  features:
    - content.code.copy # adds a copy button to every block
    - content.code.annotate # enables inline annotations

markdown_extensions:
  - pymdownx.highlight:
      anchor_linenums: true
      line_spans: __span
      pygments_lang_class: true
  - pymdownx.inlinehilite
  - pymdownx.superfences

    Basic Fenced Code Block

    Wrap code in triple backticks and add a language identifier for syntax highlighting. Pygments supports 500+ languages.

    ```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("world"))
```

    Result:

    def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("world"))

    Adding a Title

    Use the title attribute to label your code block — useful for showing a filename.

    ```yaml title="mkdocs.yml"
site_name: Su and Ivy's docs
theme:
  name: material
```

    Result:

    mkdocs.yml
    site_name: Su and Ivy's docs
theme:
  name: material

    Line Numbers

    Add linenums="1" to start numbering from line 1 (or any other number).

    ```python linenums="1"
import httpx

response = httpx.get("https://api.example.com/v1/users")
print(response.json())
```

    Result:

    1
2
3
4
    import httpx

response = httpx.get("https://api.example.com/v1/users")
print(response.json())

    Highlighting Lines

    Use hl_lines to draw attention to specific lines.

    ```python linenums="1" hl_lines="2 3"
def connect(host: str, port: int):
    if port not in (80, 443):
        raise ValueError(f"Unsupported port: {port}")
    return f"Connected to {host}:{port}"
```

    Result:

    1
2
3
4
    def connect(host: str, port: int):
    if port not in (80, 443):
        raise ValueError(f"Unsupported port: {port}")
    return f"Connected to {host}:{port}"

    Inline Code Highlighting

    With pymdownx.inlinehilite you can highlight inline snippets:

    Use `#!python print("hello")` to output text, or `#!bash export KEY=value` to set env vars.

    Result: Use print("hello") to output text, or export KEY=value to set env vars.

    Code Annotations

    Annotations let you attach explanatory notes to specific lines. Add # (1) markers in the code, then list the explanations below with 1..

    ```yaml
theme:
  name: material
  features:
    - content.code.copy # (1)!
```

1. Adds a one-click copy button to every code block on the page.

    Result:

    theme:
  name: material
  features:
    - content.code.copy # (1)!
    1. Adds a one-click copy button to every code block on the page.
